Return data about a genre (description) by genre name (e.g., “Drama”); http://localhost:8080/movies/genre/Drama GET None A JSON object with the data about a movie genre. E.g.:
{
"Name": "Drama",
"description": "A young American comes to Italy for vacation to find the key to her mother`s mystery"
}
Return data about a director (bio, birth year, death year) by name; /movies/directors/:directorName GET None A JSON object with the data about the movie director E.g.:
{
"Name": "Bernardo Bertolucci",
"Birth Date": 1941
}
Allow new users to register; http://localhost:8080/users POST A JSON object with the data about a new user. E.g.: {
"name": "Max",
"favoriteMovie": []
} 		A JSON object with the data about a new user including the id(username). E.g.:
{
username: “nickname”,
favoriteMovie: []
}
Allow users to update their user info (username); http://localhost:8080/users/:id PUT {
"name": "Maddie updated her Name",
"favoriteMovie": ["Stealing Beauty"] }		 A text message indicating that their user info (username) has been updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added); http://localhost:8080/users/:id/:movieTitle PUT None A text message indicating that a movie has been added
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed); http://localhost:8080/users/:id/:movieTitle DELETE None A text message indicating that a movie has been removed
Allow existing users to deregister (showing only a text that a user email has been removed) http://localhost:8080/users/:id DELETE None A text message indicating that a user email has been removed
